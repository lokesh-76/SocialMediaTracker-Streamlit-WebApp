# TrendPulse **Real■time social media insights & trending issue tracker**
---
## ■ What is TrendPulse? TrendPulse is a web application built with **Streamlit** that collects,
analyzes, and visualizes trending conversations from Reddit and other online platforms. It surfaces
emerging topics, sentiment shifts, and buzzworthy issues to help you stay ahead of the curve.
## ■ Why It Matters in Daily Life - **Stay informed** — Consolidate trending themes in one place. -
**Track public sentiment** — Great for creators, marketers, and researchers. - **Spot early trends** —
Discover issues before they go mainstream. - **Data-driven decisions** — Act on insight, not
guesswork.
## ■ Features - Reddit threads & comments fetcher - Trend detection: top keywords, rising topics -
Sentiment analysis / polarity tracking - Interactive graphs, word clouds, and heatmaps
## ■ Getting Started ### Installation ``` git clone https://github.com/lokesh-76/TrendPulse.git cd
TrendPulse pip install -r requirements.txt streamlit run app.py ``` Visit http://localhost:8501 to view.
## ■ How It Works 1. **Data collection** via Reddit APIs or RSS feeds 2. **Processing** using NLP
and keyword analysis 3. **Visualization** via Streamlit dashboards
## ■ Example Use Case Monitor trending discussions about AI or gaming and analyze sentiment shifts
— great for bloggers, researchers, or analysts.
## ■ Customization & Extensibility Integrate new APIs (Twitter, RSS), apply topic modeling, or
automate alerts for specific keywords.
## ■■ Limitations & Future Work - API rate limits - More advanced NLP planned
