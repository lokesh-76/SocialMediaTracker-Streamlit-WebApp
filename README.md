**Real-time social media insights & trending issue tracker**

TrendPulse is a web application built with Streamlit that collects, analyzes, and visualizes trending conversations from Reddit and other online platforms. It surfaces emerging topics, sentiment shifts, and buzzworthy issues to help you stay ahead of the curve.

## ğŸ›  Installation

### Prerequisites

- Python 3.7+
- Streamlit
- Requests
- Pandas
- Matplotlib

### Steps

1. Clone the repository:
   ```bash
   git clone https://github.com/lokesh-76/TrendPulse.git
   ```
2. Navigate into the project directory:
   ```bash
   cd TrendPulse
   ```
3. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```
4. Run the application:
   ```bash
   streamlit run app.py
   ```
5. Open your browser and go to `http://localhost:8501` to view the dashboard.
